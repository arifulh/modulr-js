Modul R

Javascript modular framework loosely based on AMD JS API Specifications.

Please see Wiki for documentation.

Demo

A demo is available with this repository. You can run a quick node server with http-server:

# install http-server
npm install http-server -g

# go to demo folder and run
cd demo
http-server -p 1337

# view on browser
http://localhost:1337/
